# Examples

See Zene in action with these real-world scenarios.

## 1. Legacy Code Refactoring
**Scenario**: Taking a messy, untyped Python script and turning it into a modular, Pydantic-based application.
[Read the Blog Post](/blog/refactoring-legacy)

## 2. Self-Healing Compiler (Rust)
**Scenario**: Fixing a Rust project with borrow checker errors. Zene runs `cargo build`, analyzes errors, and fixes them automatically.
[Read the Blog Post](/blog/self-healing-compiler)

## 3. Data Analysis
**Scenario**: Cleaning a CSV dataset and generating matplotlib charts. Zene handles missing values, invalid formats, and data visualization.
[Read the Blog Post](/blog/data-analysis)

## 4. Multi-File API
**Scenario**: Building a production-ready FastAPI backend with SQLite integration. Zene generates a modular project structure with models, schemas, and CRUD logic.
[Read the Blog Post](/blog/multi-file-api)
