Ensures a value is in the entry by inserting the default if empty, and returns a mutable reference to the value in the entry.

初步的理解是，在插入一个 Value 到 entry 的时候，如果是空的，则把 Default 插入。并且返回一个针对 entry 里的这个 Value 的可变应用。

letters 是一个 HashMap，通过 entry 获取得到针对某个字符的 entry ，然后调用 or_insert 。
