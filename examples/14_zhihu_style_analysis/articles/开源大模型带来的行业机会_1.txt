结合上面内容和我们的理解，形成了下面一个图：

从用户角度开始，用户存在将现有的专家、规则化系统转换为

大语言模型

(LLM)驱动的智能化系统的需求。在接口这一层，从原有的提示词、到最近的 Agent 和 Function Call 、Code interpreter 等工具的快速发展，为用户能够使用使用 LLM 提供接入能力

微调(Finetune)就是从通用 LLM 使用自己的数据，训练出适合自己场景的模型；而

预训练

，就是基于数据训练出通用基础 LLM

在 OpenAI 通过 ChatGPT 获取巨大领先优势之后，

开源模型

在慢慢发展。大家主要集中在 ChatGPT 的生态上发展，为了降低模型的不准确性，OpenAI 提供了 Function call 等工具，方便用户采用接口和 json 形式和 ChatGPT互动

但是大量用户存在使用自己数据在本地训练自己模型赋能业务的需求。完全把自己数据交给 Openai，对于中大型企业来说都不太能接受。最近，Stability AI的CEO说

目前该技术还处于发展的早期阶段

，“AI还没有完全准备好在

金融服务

等大型行业大规模部署，但我们可以看到它的价值。”

但是无论 ChatGPT 怎么优化体验，如果不能实现数据的自己控制。大型企业，尤其是

金融企业

，是不会把主要流程转移到 ChatGPT

在巨大的本地训练需求下，开源大模型获得了大家的期待。并且随着 Llama2 的推出，大家对使用开源大模型赋能本地业务又产生了极大的热情

但是门槛也很大。因为现有的

企业架构

是以专家经验和规则流程为核心。AI 的不准确性导致企业不能完全信任 AI 来确定工作流程和关键决策。所以一个通常的想法是，在不重要的场合， 用 AI 来辅助员工提高效率，错误检查还是需要企业员工来负责

这样 AI 应用方向就会产生两个分支：一个是 AI 公司希望提高能力来适配企业的关键流程；一个是适配企业员工的需求，例如生成报告片段、生成 SQL、调取 API

但是不管是什么方式，都很难绕过两个关键因素，训练和微调

从数据，经过训练，形成基础模型。然后再次训练，或者微调，形成针对某个场景的场景模型。这个过程是提高适配企业场景能力的必由之路。

训练是一个研究过程，大多数不针对特定场景，或者说，拿特定场景重新训练的成本太高，造成大家不愿意投入。这有点像

云服务

，自己大家一个简单的云服务架构对企业来说还算可行。但是

云计算技术

还在发展，产品越来越多。企业很难有长期的研发团队跟踪技术，完善产品，成本过高。很容易自己忙了半天，产业技术一提升就落后了。

模型也是，Llama2 只是掀起来开源模型的第一波。还会有更多的模型出现，Llama2 也会升级到 Llama3。

从投资角度，模型的训练以及相关训练产业链是一个

投资回报

非常长期的过程。就像云服务，需要建立足够的技术、产品和商业服务壁垒，形成和开源模型相比高出一大截的优势，加上服务，来吸引客户采纳和付费。云服务直到现在能够实现盈利的还只有

亚马逊

和微软，说明这个模式门槛极高，

投资收益

的回报率很长。

微调实际上是在通用模型的基础上用数据和场景来二次训练通用模型，提高其对场景应用的适配能力。有点像基于原厂CRM 或者 ERP 软件，针对企业进行

二次开发

和实施的过程。

为什么会有预训练和微调的区分，主要还是成本原因。由于大模型参数巨大，如果每次针对场景应用的调整都针对全部参数进行计算，成本巨大。通过冻结某一些参数，流出一些参数来训练，会减少成本。当然，这样做也会有损失。在这个路径上，类似 Lora 和 Qlora 的方法也在平衡成本和效果，也会有更多其他的方法出现

不管是训练和微调，都有一个原材料，就是数据。因为大语言模型广泛采用强化学习方法，还有一些奖励模型的方式。所以原始数据到能够进行训练或者微调，还有一个转换过程。

根据以上的分类，罗列了一下做这几块业务的初创公司

随着 Llama2 的发布，以及更多新的好的开源模型出现，基于开源模型的训练、微调和应用会蓬勃的发展起来。今天 Stability AI 又发布了 DeepFloyd 和 CarperAi 的 FreeWilly2 ，评测结果已经超过了 GPT-3.5。另外，应用端随着 AI Agent 模式为 LLM 提供了外部数据的支持，让用户更好的用自己数据结合模型推理能力来做场景，应该会得到更好的效果。应用也会加快发展起来。

还有就是数据，存在一些大型企业系统训练自己的模型，用自己的数据。对

行业数据

及

企业数据

的整理也会有更多的需求，甚至企业数据的处理方式都可能有较大的变化，对数据，尤其是非结构化数据处理平台有较大的推动作用

以上是针对 Llama2 出来之后的一些思考。AI 发展又进入了一波每日新气象的时代。效果较好的开源大模型带来了更多的想象空间，让我们拭目以待
